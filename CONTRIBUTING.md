# 🤝 Contributing to LLM Learning Hub

Thank you for your interest in contributing! This is a personal learning repository, but suggestions are welcome.

## 📝 How to Contribute

### Reporting Issues
- Use the issue templates
- Provide clear reproduction steps
- Include environment details

### Adding Content
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-topic`
3. Follow the existing structure
4. Add a README.md in new folders
5. Submit a pull request

### Code Standards
- Use clear variable names
- Add comments for complex logic
- Include docstrings for functions
- Keep notebooks runnable top-to-bottom

## 📁 File Naming
- Notebooks: `XX-descriptive-name.ipynb`
- Scripts: `descriptive_name.py`
- Docs: `descriptive-name.md`

## 🏷️ Commit Messages
```
feat: add transformer attention visualization
fix: correct gradient calculation in backprop
docs: update RAG section with new methods
```
