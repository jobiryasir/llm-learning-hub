# Recommended Courses

## Free Online Courses

### Deep Learning Foundations
- [Fast.ai - Practical Deep Learning for Coders](https://course.fast.ai/)
  - Top-down approach to deep learning
  - Free, hands-on, and practical
  - Covers CNNs, RNNs, and modern techniques

- [DeepLearning.AI - Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction)
  - Andrew Ng's foundational course
  - Supervised and unsupervised learning
  - Neural networks and deep learning

- [MIT 6.034 - Artificial Intelligence](https://ocw.mit.edu/courses/6-034-artificial-intelligence-fall-2010/)
  - MIT OpenCourseWare
  - Classic AI concepts and algorithms

### NLP and Transformers
- [Hugging Face NLP Course](https://huggingface.co/learn/nlp-course)
  - Free course on NLP with Transformers
  - Tokenizers, models, and pipelines
  - Fine-tuning and deployment

- [Stanford CS224N - Natural Language Processing with Deep Learning](https://web.stanford.edu/class/cs224n/)
  - Stanford's NLP course
  - Word vectors, RNNs, transformers
  - Lecture videos and assignments

- [Stanford CS324 - Large Language Models](https://stanford-cs324.github.io/winter2022/)
  - Stanford's LLM course
  - Pretraining, fine-tuning, alignment
  - Comprehensive LLM overview

### LLM-Specific Courses
- [DeepLearning.AI - Generative AI with Large Language Models](https://www.coursera.org/learn/generative-ai-with-llms)
  - AWS and DeepLearning.AI collaboration
  - Pretraining, fine-tuning, RLHF
  - Practical labs

- [DeepLearning.AI - Finetuning Large Language Models](https://www.deeplearning.ai/short-courses/fine-tuning-large-language-models/)
  - Short course on fine-tuning
  - LoRA and parameter-efficient methods

- [DeepLearning.AI - Reinforcement Learning from Human Feedback](https://www.deeplearning.ai/short-courses/reinforcement-learning-from-human-feedback/)
  - RLHF fundamentals
  - Reward modeling and PPO

- [DeepLearning.AI - Building Systems with the ChatGPT API](https://www.deeplearning.ai/short-courses/building-systems-with-chatgpt/)
  - Building applications with LLMs
  - Classification, moderation, chaining

- [DeepLearning.AI - LangChain for LLM Application Development](https://www.deeplearning.ai/short-courses/langchain-for-llm-application-development/)
  - LangChain fundamentals
  - Chains, agents, and memory

- [DeepLearning.AI - Vector Databases: from Embeddings to Applications](https://www.deeplearning.ai/short-courses/vector-databases-embeddings-applications/)
  - Vector databases and RAG
  - Embeddings and similarity search

- [LLM University (Cohere)](https://cohere.com/llmu)
  - Cohere's free LLM course
  - Text generation, embeddings, classification

- [Google Generative AI Course](https://www.cloudskillsboost.google/paths/118)
  - Google's official GenAI course
  - Vertex AI and Gemini

- [Full Stack LLM Bootcamp (Full Stack Deep Learning)](https://fullstackdeeplearning.com/llm-bootcamp/)
  - Free LLM bootcamp
  - Prompt engineering, fine-tuning, deployment

- [LLM Foundations (Moonshot AI)](https://www.moonshot.cn/)
  - Moonshot AI's LLM course
  - Chinese and English content

### Specialized Topics

- [DeepLearning.AI - Multi AI Agent Systems with CrewAI](https://www.deeplearning.ai/short-courses/multi-ai-agent-systems-with-crewai/)

- [Hugging Face - Agents Course](https://huggingface.co/learn/agents-course/)
  - Building AI agents
  - smolagents and tool use

- [Hugging Face - RAG Cookbook](https://huggingface.co/learn/cookbook/en/rag)
  - Retrieval-Augmented Generation
  - End-to-end implementation

## Books

### Must-Read
- [*Build a Large Language Model (From Scratch)* - Sebastian Raschka](https://www.manning.com/books/build-a-large-language-model-from-scratch)
  - Manning Publications, 2024
  - Implement GPT from scratch
  - Pretraining, fine-tuning, and alignment

- [*Hands-On Large Language Models* - Jay Alammar & Maarten Grootendorst](https://www.oreilly.com/library/view/hands-on-large-language/9781098150961/)
  - O'Reilly Media, 2024
  - Practical guide to LLMs
  - LangChain, LlamaIndex, and more

- [*Natural Language Processing with Transformers* - Lewis Tunstall, Leandro von Werra, Thomas Wolf](https://www.oreilly.com/library/view/natural-language-processing/9781098136781/)
  - O'Reilly Media, 2022
  - Hugging Face authors
  - Transformers from scratch

- [*Understanding Deep Learning* - Simon J.D. Prince](https://udlbook.github.io/udlbook/)
  - MIT Press, 2023
  - Free PDF available
  - Comprehensive deep learning theory

- [*Deep Learning* - Ian Goodfellow, Yoshua Bengio, Aaron Courville](https://www.deeplearningbook.org/)
  - MIT Press, 2016
  - Free online book
  - The deep learning bible

- [*Neural Networks and Deep Learning* - Michael Nielsen](http://neuralnetworksanddeeplearning.com/)
  - Free online book
  - Step-by-step implementation

- [*Mathematics for Machine Learning* - Marc Peter Deisenroth, A. Aldo Faisal, Cheng Soon Ong](https://mml-book.github.io/book/mml-book.pdf)
  - Cambridge University Press, 2020
  - Free PDF available
  - Math foundations for ML

- [*The Hundred-Page Machine Learning Book* - Andriy Burkov](http://themlbook.com/)
  - 2019
  - Concise ML overview

- [*Designing Machine Learning Systems* - Chip Huyen](https://www.oreilly.com/library/view/designing-machine-learning/9781098107959/)
  - O'Reilly Media, 2022
  - ML engineering best practices

- [*Building Machine Learning Pipelines* - Hannes Hapke, Catherine Nelson](https://www.oreilly.com/library/view/building-machine-learning/9781492053187/)
  - O'Reilly Media, 2020
  - TensorFlow Extended (TFX)
